# Cava Compiler

Implementation of a Lexer and Parser for the course CS F363 Compiler Construction.

Dependencies:

-   C++ boost library
-   pip graphviz package
-   graphviz executables in path

To Parse:

-   make
-   build/parse is the executable
-   visualize the parse tree using graph.py
